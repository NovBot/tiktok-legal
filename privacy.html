Política de Privacidad

Última actualización: 2026

Esta Política de Privacidad describe cómo se recopila, utiliza y protege la información en relación con la aplicación “TikTok Automation”, que se utiliza exclusivamente para automatizar la subida de vídeos a TikTok mediante la API oficial de TikTok.

1. Información que recopilamos

La aplicación no recopila información personal de usuarios finales. Solo procesamos datos técnicos estrictamente necesarios para funcionar:

  • Tokens de acceso generados por TikTok (proporcionados por el usuario propietario de la app).
  • Archivos de vídeo generados automáticamente y enviados a TikTok.

No almacenamos, vendemos ni compartimos información personal con terceros.

2. Cómo usamos los datos

Los datos proporcionados se utilizan únicamente para:

  • Generar vídeos automáticamente a través de servicios propios.
  • Subir dichos vídeos a la cuenta de TikTok autorizada por el titular.

No se realiza ningún otro procesamiento ni análisis con los datos.

3. Cómo protegemos la información

La comunicación con TikTok se realiza usando HTTPS y los tokens se usan únicamente para la autenticación necesaria dentro del flujo automatizado. No se almacenan permanentemente.

4. Servicios de terceros

La app utiliza únicamente la API oficial de TikTok. No compartimos datos con proveedores externos.

5. Derechos del titular de la cuenta de TikTok

El titular puede revocar en cualquier momento el acceso desde TikTok Developers y eliminar cualquier token asociado.

6. Contacto

Para preguntas sobre esta política, puedes contactar al desarrollador a través de GitHub.
