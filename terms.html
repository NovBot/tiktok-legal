Términos de Servicio

Última actualización: 2026

Estos Términos de Servicio regulan el uso de la aplicación “TikTok Automation”. La aplicación está diseñada exclusivamente para subir vídeos generados automáticamente a una cuenta de TikTok autorizada por su propietario.

1. Uso permitido

El usuario se compromete a utilizar la aplicación únicamente para propósitos legales y cumpliendo las políticas de TikTok Developers. La automatización solo se realiza sobre la cuenta TikTok que autoriza el proceso.

2. Propiedad del contenido

Todo contenido (vídeos, imágenes, audio) subido a TikTok pertenece exclusivamente al titular de la cuenta. La aplicación no reclama propiedad ni derechos sobre dicho contenido.

3. Limitaciones de responsabilidad

La aplicación se proporciona “tal cual”, sin garantías de ningún tipo. El desarrollador no es responsable de:

  • Problemas derivados del uso de la API de TikTok.
  • Interrupciones del servicio.
  • Eliminación o rechazo de contenido por parte de TikTok.

4. Integración con TikTok

La función principal es automatizar la subida de contenido mediante la API oficial. Cualquier cambio en la API de TikTok puede afectar el funcionamiento.

5. Cancelación

El usuario puede dejar de usar la aplicación en cualquier momento. También puede revocar los tokens directamente desde TikTok Developers.

6. Modificaciones

Nos reservamos el derecho de actualizar estos términos cuando sea necesario. Las actualizaciones estarán visibles en esta página.

7. Contacto
